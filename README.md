## Key Features
* **Advanced Geostatistics:** Includes QA/QC control charts, normality tests, and multivariate regression for geochemical data.
* **Integrated Cartography:** Automated generation of drillhole plan views, cross-sections, and spatial distribution maps.
* **Stereographic Analysis:** Built-in structural module for plotting and analyzing orientation data (dip/strike), identifying joint sets, and performing kinematic stability analysis.
* **Drillhole Data Audit:** Comprehensive validation of DH databases against industry standards.

## Technical Specifications
* **Core Engine:** Python 3.12+
* **Visualization:** High-performance 2D/3D rendering for structural and spatial data.
* **Structural Toolset:** Equal-area and equal-angle stereonet projections.
